# JAVA_DICTIONARY_APP

## Biên dịch chương trình

Từ thư mục gốc của dự án, chạy lệnh sau trong **Command Prompt (Windows)** hoặc **Terminal (Linux/macOS):**

```bash
javac -d out Main.java
```

## Sau khi chạy lệnh này, một thư mục `out/` sẽ được tạo chứa toàn bộ file `.class`.

## Chạy chương trình

Sau khi biên dịch xong, chạy lệnh:

```bash
java -cp out Main
```
